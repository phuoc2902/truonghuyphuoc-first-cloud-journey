**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Initialize Next.js App Router project with TypeScript and set up base folder structure.](1.1-week1/)

**Week 2:** [Configure Axios interceptors for distributed Request ID tracing and automatic JWT token handling.](1.2-week2/)

**Week 3:** [Implement Global State Management for Auth and develop structural Layout Skeletons.](1.3-week3/)

**Week 4:** [Implement Frontend Cart Logic with Local Storage capability for guest shoppers.](1.4-week4/)

**Week 5:** [Develop Backend Cart REST APIs using Spring Boot and relational database design.](1.5-week5/)

**Week 6:** [Implement client-to-server Cart Synchronization logic upon successful user login.](1.6-week6/)

**Week 7:** [Install, configure, and initialize Message Queue (RabbitMQ/Kafka) infrastructure.](1.7-week7/)

**Week 8:** [Implement Message Publisher pattern for asynchronous email and system notifications.](1.8-week8/)

**Week 9:** [Develop Message Consumers to process asynchronous email sending and notification delivery.](1.9-week9/)

**Week 10:** [Write comprehensive Unit and Integration tests to validate codebase stability.](1.10-week10/)

**Week 11:** [Design and configure GitLab CI/CD Pipeline for automated integration.](1.11-week11/)

**Week 12:** [Automate Docker packaging and push release images to GitLab Container Registry.](1.12-week12/)
